Detailed Description

Upload doxygen documentation to GitHub pages.

Author:
Dan Fekete <thefekete@gmail.com>

Clones master branch from github into a temp directory and runs doxygen. Once all the doxygen docs are created, it checks out gh-pages branch (deletes it if it exists on github!), rm's all the files except docs, then pushes to github. Finally, it deletes the temp directory.
